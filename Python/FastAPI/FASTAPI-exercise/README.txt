Como executar a api ou test

primeramente instale o requerimentos usando no terminal : "pip install -r requirements.txt"


Ligando a API:
    no diretorio escreva no terminal "uvicorn main:app", isso criara um servidor em "http://127.0.0.1:8000/verify",
    e agora so testar a api

Testando:
    no diretorio escreva no terminal "pytest test.py", e com isso executara o testes
